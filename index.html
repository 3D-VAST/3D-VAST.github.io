2. Topic

Core Challenges

  • Sparse and incomplete views
  • Visual ambiguities across altitudes
  • Spatio-temporal inconsistencies
  • Dynamic scene changes

Research Directions

  1. Cross-altitude feature matching
  2. View synthesis from heterogeneous data
  3. Sparse-view 3D reconstruction
  4. Generative view completion
